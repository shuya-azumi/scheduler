# 面接 日程調整サービス（MVP）

人材紹介の3者（求職者・エージェント・事業所）が、**チャット形式で面接日程を調整**するための最小プロトタイプ。
候補日時をボタンで提示し、クリック＝チャットに流れる。全員が「行ける」を選んだ枠が緑にハイライトされ、エージェントが確定する。

## 技術
- HTML / CSS / JavaScript（jQuery）
- Firebase **Realtime Database**（v9 モジュール・CDN読み込み）

## 使い方
1. `index.html` をブラウザで開く
2. 名前と立場（求職者／エージェント／事業所）を選んで参加
3. 同じURL（`?room=...`）を3者で共有すると同じルームに入れる
4. エージェントが候補枠を追加 → 各自が ⭕🔺❌ で回答 → 全員⭕で確定

## セットアップ（自分のFirebaseで動かす場合）
`index.html` 内の `firebaseConfig` を自分のプロジェクトの値に差し替える。
Realtime Database を作成し、検証中はルールをテストモードにする。

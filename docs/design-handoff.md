## 2026-06-13 投票ラベル変更＋Apple風リデザイン

### ① 投票ラベルを 3択→2択 に変更（※ `<script>` に触れています）
求職者・事業所の投票表現を変更しました。**内部の値（`ok` / `ng`）とデータ構造は不変**で、表示テキストだけの修正です。

- `index.html` の `renderSlots()` 内の投票ボタン：
  - `⭕行ける / 🔺迷う / ❌無理`（3択）→ **`参加可` / `不可`（2択）** に変更。
  - **`maybe`（迷う）ボタンを廃止**しました。
- `vote()` 内のチャット流し込み文言：`ok→「⭕ 参加可」` / `ng→「❌ 不可」` に変更。

⚠️ 機能セッションへの確認事項：
- `maybe` を廃止したので、今後 `votes` に `maybe` が新規に書き込まれることはありません。
  ただし**過去データに `maybe` が残っている可能性**はあります。`renderSlots()` の○×表示は
  `v === "maybe" ? "🔺"` のフォールバックを**あえて残してあり**、旧データも壊れず表示されます。
- データ設計（`docs/データ設計.md`）の vote 値の説明を 2値（ok/ng）に更新するかは機能セッション判断でお願いします。

### ② 見た目を Apple 風にリデザイン（`css/style.css` のみ）
- 配色を Apple system 色基調に（背景 `#f5f5f7` / 文字 `#1d1d1f` / ヘアライン）。
- **意味色を2系統に整理**：
  - 🟢 **緑** = `参加可` / `全員参加可`（合意状態）
  - 🔵 **ブルー** = `確定`アクション・`確定`状態（バナー／バッジ／確定ボタン／確定チャット通知）
- 投票ボタン＝iOS風カラー塗りボタン（並び順 `:nth-child` で 1=緑/2=赤を割当。**JSは不変**）。
- チャット＝iMessage 風バブル（自分=ブルー／相手=グレー／角丸18px）。
- デザイントークンは `:root` のCSS変数に集約（色変更はそこだけ）。

## 2026-06-13（追補2）役割の可視化（位置移動＋アイコン）

3者の判別性向上。配色は変更なし。

### HTML / `<script>`（※表示・アイコン挿入のみ・ロジック不変）
- **「あなた」表示を `header` から `候補日時` の上の `.me-bar` へ移動**（誰の画面か明示）。
  併せてパネル群（`.layout`）が少し下がる。`#me-label` の id は維持し、JSは `.html()` で
  アイコン＋名前＋役割を描画。
- **役割アイコン（インラインSVGの線アイコン）を追加**：求職者＝人 / エージェント＝ブリーフケース /
  事業所＝建物。`<script>` 冒頭に `ROLE_ICON` と `roleIcon(role)` を定義し、以下に挿入：
  - 参加画面の役割選択ボタン（静的HTML）
  - 「あなた」表示（`#me-label`）
  - 投票チップ（`renderSlots` の `.vote-chip`）… 各参加者の役割が分かる
  - チャットの役割タグ（`appendMessage` の `.role-tag`）
- アイコンは `currentColor` 継承なので**配色は不変**（黒バブル内では自動で白）。

### `css/style.css`
- `.role-ico`（基底サイズ＋文脈別サイズ）、`.me-bar` / `.me-label`（識別チップ化）、

## 2026-06-13（追補3）役割カラー分け＋アプリのブランディング

### 役割カラー（3者を一目で判別）
- `:root` に `--role-candidate`（緑 #16a34a）/ `--role-agent`（オレンジ #ea580c）/
  `--role-facility`（赤 #dc2626）を定義。
- 役割アイコン（`roleIcon()` のSVG・参加ボタンのSVG）に `role-candidate/agent/facility`
  クラスを付与し、`currentColor` で色分け。投票チップ・チャットの役割タグ・「あなた」表示すべてに反映。
- 「あなた」の役割タグは役割カラーの文字＋淡いタグ背景（`color-mix`）。

### アプリのブランディング（参加画面トップ＆ヘッダー）
- アプリアイコン＝**カレンダー＋チェック**のインラインSVG（面接日程調整らしさ）。
  墨色の角丸バッジに白アイコン。`.brand`（参加画面・48px）/`.brand-icon--sm`（ヘッダー・32px）。
- 参加画面の見出しを **26px / 太字700** に拡大、ヘッダーのアプリ名を **18px / 700** に。
- 画像アセットではなくSVGを採用（外部依存を増やさない・高解像度で滲まない方針）。

### 文字サイズ（追補2の続き）
- 役割名・各種ラベルの小ささを是正（役割ボタン15px太字、見出し17px、候補ラベル17px 等）。
